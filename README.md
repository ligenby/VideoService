# LensEra — Онлайн-сервис для просмотра видеоконтента

**LensEra** — это полнофункциональный клиент-серверный видеохостинг, разработанный в рамках курсового проекта. Платформа позволяет пользователям загружать мультимедийный контент, взаимодействовать с аудиторией (комментарии, оценки, подписки) и управлять собственными каналами. 

---

## Стек технологий

* **Backend:** PHP 8, SQL
* **База данных:** MySQL (Сложная реляционная структура из 23 таблиц)
* **Frontend:** HTML5, CSS3, JavaScript (AJAX для асинхронных запросов)
* **Архитектура:** Клиент-серверная (MVC-подобное разделение логики и шаблонов)
* **Среда / Сервер:** Apache (XAMPP / Open Server)
* **Дизайн:** Разработан с нуля в Figma (часть позаимстванно с YouTube)

---

## Ключевые технические решения

* **Сложная архитектура БД:** Спроектирована логическая и физическая модели данных (IDEF1X). База включает связи «многие-ко-многим», внешние ключи и каскадные обновления (таблицы видео, пользователей, категорий, плейлистов, реакций, истории и т.д.).
* **Асинхронность (AJAX):** Постановка лайков, подписка на каналы и подгрузка комментариев реализованы без перезагрузки страницы средствами JavaScript.
* **Безопасность:** Реализовано хеширование паролей, защита от CSRF-атак, безопасная система сессий и авторизации через токены («Запомнить меня»).
* **Файловая система:** Реализован безопасный механизм загрузки мультимедийных файлов (видео и обложек) на сервер с проверкой форматов и размеров.

## Роли и функционал

Проект поддерживает 3 уровня доступа с разграничением прав:

### 1. Гость (Неавторизованный пользователь)
* Просмотр главной страницы и ленты рекомендаций.
* Поиск видео по названию и ключевым словам.
* Просмотр видеороликов, каналов авторов и категорий (Музыка, Игры, Спорт и др.).
* Регистрация нового аккаунта.

### 2. Пользователь (Авторизованный автор/зритель)
* **Студия контента:** Загрузка новых видео, установка превью (обложек), добавление метаданных (название, описание, категория, настройки приватности).
* **Социальное взаимодействие:** Оставление комментариев (включая древовидные ответы), система лайков/дизлайков, подписка на другие каналы.
* **Управление профилем:** Оформление канала (аватар, баннер), настройка ссылок.
* **Организация контента:** Создание плейлистов, добавление видео в список «Смотреть позже», сохранение истории просмотров.
* **Уведомления и модерация:** Получение системных уведомлений (новые подписчики, лайки), отправка жалоб на неприемлемый контент.

### 3. Администратор (Панель управления)
* **Модерация контента:** Принудительное удаление видео и комментариев, нарушающих правила платформы.
* **Управление пользователями:** Просмотр списка всех аккаунтов, блокировка/разблокировка нарушителей.
* **Система тикетов:** Обработка жалоб пользователей и ответы на отзывы о работе сервиса.
